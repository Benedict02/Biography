Biography

Stuff that I learn from CaniComp

4-5 weeks ago, we started learning HyperText Markup Language(HTML) and Cascading Style Sheets(CSS) in CaniComp. Even though I learned HTML and CSS since grade 6, reviewing things that I learned and knowing new element/attribute in HTML is really fun. So week 1 up until week 3, we learned about elements and attributes in HTML using this great website called FreeCodeCamp. After learning some basics in HTML, we learned CSS. CSS is a framework used to beautify a HTML webpage, it's like plastic surgery for HTML. In my opinion CSS is pretty straightforward, but you need creativity and a lot of attribute implementation in order to make the webpage fabulous. I don't think I have anything more to talk about, we're just learning HTMl and CSS so there isn't really much to talk about.


Due to lack of topic for this Bio, I would like to talk about how this webpage is created. This project started around Thursday, 24th February 2022. So the assignment is making a webpage using HTML and CSS, the topic is biograpy about stuff that I learned over the past month(s). I use HTML to make the text, paragraph, heading, icons, buttons, links, and the translate UI. After programming the HTML file, I linked a CSS file externally using the link element. Inside of the .css file, there are a lot of stuff happening. I divide some elements into classes, using those classes I can make the .css file change how the HTML element should be displayed. I realize that using HTMl and CSS is really easy for me. So I want to implement JavaScript, I added light theme into the webpage and even a "translate page" selection using Google Translate API. I learned some JavaScript basics arround 2-3 days, I personally dislike learning JavaScript because unlike python the syntax is a bit complicated.
