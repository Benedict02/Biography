Stuff that I learn from CaniComp

4-5 minggu yang lalu kita mulai belajar materi HyperText Markup Language(HTML) dan Cascading Style Sheets(CSS) di eskul CaniComp. Walaupun saya sudah belajar HTML dan CSS dari kelas 6, mereview ulang dan mempelajari element/attribute baru di HTML sangat menyenangkan. Pada minggu pertama hingga minggu ketiga kita mempelajari element dan attribute HTML di FreeCodeCamp. Setelah mempelajari basic element HTML, kita mempelajari CSS. CSS adalah framework yang digunakan untuk memperindah webpage HTML, anggap aja seperti operasi plastik untuk HTML.
